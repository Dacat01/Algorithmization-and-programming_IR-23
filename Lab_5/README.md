# Алгоритм Кнута-Морріса-Прата

Для заданої стрічки знайти всі місця входження підстрічки. 	


Вхідні дані:
	стрічка, підстрічка

Вихідні дані:
	всі початкові місця входження пістрічки у стрічку, та -1 якщо підстрічка є пустою або входжень у стрічку немає
	
	
Приклад:

In:
AABBABAA, AB

Out:
1 4


In:
AABBABAA, abc

Out:
-1
